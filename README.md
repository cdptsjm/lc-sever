# lc-sever
-领创标准服务端代码
-使用php+js+json编写的领创后端和网络服务端
-需要在服务器环境上安装，githubpage不行

# 介绍
通过模仿领创返回值的形式操控平板
可以自定义功能，应用等
可以用于解除控制或保存文件

# 安装
在服务器环境中git本仓库，并创建网站

# 使用
- 解控/接管
设置网站端口为883,
在计算机上使用以下命令添加host(win管理员),并开启热点
```
echo cloud.linspirer.com <yoururl(远程)/127.0.0.1(本地)> >>c:\windows\system32\etc\hosts
```
在网页端创建账号，账号email=你的领创账号,其他随便填
使用平板连接热点并刷新策略即可在网页端操控

- 全新安装
如果你需要在其他设备使用lc服务/桌面则:
克隆本仓库并创建网站，假设网站是http://123.12.123.123:883
则lc启动时连点7次登录框上方空白处，
网址填
```
<http/https>://<your url>:883/
```
如示例，将类型改为http,框内只需填url(无需http/https)
```
123.12.123.123:883
```
在网页端创建账号并登录即可
